# Trading Bot Pro - Standalone Version 
 
## 🚀 How to Use 
 
1. Make sure Node.js is installed on your computer 
   Download from: https://nodejs.org/ 
 
2. Double-click "TradingBotPro.bat" to start the application 
 
3. The first time you run it, it will install dependencies automatically 
 
4. Enjoy your professional trading bot! 
 
## 📋 Features 
- Professional desktop interface 
- Real-time trading simulation 
- Multiple trading strategies 
- Performance tracking 
- Risk management tools 
 
## ⚠️ Requirements 
- Windows 10 or later 
- Node.js installed 
- Internet connection for live trading
